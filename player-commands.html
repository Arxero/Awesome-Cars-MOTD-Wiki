Player Commands

Name Use Description
Server menu say_team menu Opens the server menu.
Nominate menu say_team nomenu A menu with all maps in the server from which you can nominate maps.
Unstuck say_team /unstuck Unstuck yourself.
Mute players say_team /mute Mute annoying players.
Rock the vote say_team rtv Initiate map change with your vote.
Boost menu say_team boost Opens a menu with all information for boosting the server.
Rank say_team /rank Shows your rank.
RULES say_team /rules Player rules.
Online Admins say_team /admins Shows all online admins and vips.
Roundsound say_team /roundsound Turns off the music at end round.
RTD say_team rtd Roll the dice and win something cool.
GtLink say_team /gtlink Shows your gametracker.com link.
Napalm say_team /napalm Gets you a napalm grenade.
bipbip say_team bipbip Bipbip sound effect.
